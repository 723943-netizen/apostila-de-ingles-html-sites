# 📚 Apostila de Inglês - Regime de Exercícios Domiciliares (RED)

Um site interativo para o apoio pedagógico ao ensino de Inglês no 1º ano do Ensino Médio.

## ✨ Características

✅ **Responsivo** - Funciona perfeitamente em smartphone, tablet e desktop
✅ **Interativo** - Responda diretamente no navegador
✅ **Offline** - Salva progresso automaticamente no navegador
✅ **Imprimível** - Exporte para PDF com formatting profissional
✅ **Gabarito Protegido** - Liberado após envio das respostas
✅ **Mobile-First** - Otimizado para dispositivos móveis

## 🚀 Como Usar

### Online
1. Acesse o site pelo GitHub Pages
2. Preencha seus dados no cabeçalho
3. Responda as questões no navegador
4. Seu progresso é salvo automaticamente
5. Clique em "📊 Relatório & Gabarito" para enviar

### Offline
1. Clone este repositório
2. Abra `index.html` no navegador
3. Use normalmente - funciona sem conexão!

## 🎯 Conteúdo Disponível

### Unidade 1: Pronomes Sujeito e Objeto
- Subject Pronouns (I, You, He, She, It, We, They)
- Object Pronouns (me, you, him, her, it, us, them)
- Atividades de fixação

### Unidade 2: Presente Contínuo
- Present Continuous (am/is/are + -ING)
- Formas afirmativa, negativa e interrogativa
- Exercícios práticos

### Unidade 3: Conectivos
- Linking Words (and, but, although, because, so)
- Categorias: adição, contraste, causa & efeito
- Exercícios de integração

### Unidade 4: Preposições de Tempo
- In, On, At - quando usar cada uma
- Contextos de uso
- Atividades de fixação

### Avaliação Final
- Parte I: 10 questões objetivas (5.0 pontos)
- Parte II: Questões discursivas (5.0 pontos)
- Gabarito de autocorreção

## 💾 Armazenamento de Dados

Todas as respostas são armazenadas no **localStorage** do navegador:
- Chave: `red_english_workbook_blank_print_v2_secure`
- Dados salvos: inputs, radios, status do envio
- Sem conexão com servidor - privacidade garantida!

## 🔐 Segurança

- Gabarito codificado em Base64 (proteção anti-spoiler)
- Liberação manual do gabarito após envio
- Email institucional para relatórios: `723943@profe.sed.sc.gov.br`

## 📧 Envio de Relatórios

Os relatórios são enviados por email (cliente de email do usuário):
- Nota parcial da Parte I (objetivas)
- Compilação de todas as respostas
- Timestamp do envio
- Dados do aluno

## 🖨️ Impressão

Clique em "Gerar PDF / Imprimir" para:
- Limpar automaticamente as respostas digitadas
- Manter espaços em branco para preenchimento manual
- Formatar com linhas pontilhadas (padrão de apostila)
- Restaurar respostas após impressão
